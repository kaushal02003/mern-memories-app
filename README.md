# Memories Social Platform | Full-Stack MERN Ecosystem

### **Project Overview**
Architected a high-performance, full-stack social media application designed for seamless digital storytelling. This project features a **decoupled architecture**, bridging a reactive frontend with a robust, scalable backend to manage complex user-generated content in real-time.



### **Core Technical Contributions**
* **End-to-End Development:** Engineered the entire Software Development Lifecycle (SDLC), from NoSQL schema design to front-end component architecture.
* **State Management:** Implemented **Redux** for global state synchronization, ensuring consistent data flow across the application and optimizing rendering performance.
* **RESTful API Design:** Developed a secure API layer using **Node.js and Express** to handle multi-variable CRUD operations and complex data filtering.
* **Responsive UI/UX:** Utilized **Material-UI** and advanced CSS patterns to deliver an exceptional, mobile-first user experience.

### **Technical Stack**
* **Frontend:** React.js, Redux, Material-UI, Axios
* **Backend:** Node.js, Express.js
* **Database:** MongoDB Atlas (NoSQL)
* **Tools:** Postman, Git, Vercel
